# GREYN Frontend

This is the frontend application for the GREYN Sustainable Micro-Investment Platform.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4

## Features

- ✅ Complete GREYN brand kit implementation
- ✅ Professional wordmark logo
- ✅ Helvetica Now Bold for headings
- ✅ Poppins for body text
- ✅ Brand color palette
- ✅ Custom favicon

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel with automatic deployments from the main branch.
